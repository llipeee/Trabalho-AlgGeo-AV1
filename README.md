## 📂 Estrutura do Projeto

- `matrix.py` → implementação da classe Matrix  
- `vector.py` → implementação da classe Vector  
- `linear_algebra.py` → operações e algoritmos  
- `main.py` → demonstração das funcionalidades  
- `tests.py` → testes automatizados  

---

## 🧠 Funcionalidades

- Criação e manipulação de matrizes e vetores  
- Soma e multiplicação (escalar, elemento a elemento e matricial)  
- Transposição  
- Eliminação de Gauss  
- Método de Gauss-Jordan  
- Resolução de sistemas lineares
